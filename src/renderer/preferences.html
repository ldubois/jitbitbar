Connexion Jitbit

Renseigne ton helpdesk et ton token d'API pour suivre tes demandes.

L'adresse racine, l'API est sous /api automatiquement.
Jitbit → ton avatar → « Token API ». Laisse vide pour conserver le token déjà enregistré.

Connexion Asana

Pour transformer une demande en tâche Asana en un clic.

Asana → Settings → Apps → Personal access tokens. Laisse vide pour conserver le token enregistré.
